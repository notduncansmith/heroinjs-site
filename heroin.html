HeroinJS

A dependency injection framework for Node.js and the browser

Installation:

npm install heroin

Usage:

//Register a dependency h.registerWithFactory('myRepo', function() { //create and return your repository here });


//register an instance var factoryRepo = new FactoryRepo();
h.registerWithInstance('myOtherRepo',factoryRepo);


//resolve a registered factory or instance var repo = h.resolve('myRepo')
, otherRepo = h.resolve('myOtherRepo');

FAQ:

  • Why use Heroin?

    Don't use heroin!

  • Sorry, why use Heroin.js?

    Managing dependencies sucks. Heroin.js makes it suck less.

  • How can I contribute?

    Just fork the repo, make your changes, and submit a pull request.

  • Where are the unit tests?

    We didn't feel like writing them. However, you're more than welcome to contribute some!
